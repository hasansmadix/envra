# Contributor Covenant Code of Conduct

We follow the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) (v2.1).

**Summary:** be respectful, welcome newcomers, disagree professionally, and prioritize the safety of the community.

For questions or reports, contact the project maintainers via the channels listed on the GitHub repository.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
